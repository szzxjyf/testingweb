AMI Performance Tracking Tool - User Guide

Overview

This tool processes three CSV files and generates comprehensive performance statistics by staff and month. All processing is done locally in your browser - no data is uploaded to any server.

Required Files

File 1: SMT Data

Required Columns:

  • Month ID - Format: 6 digits (e.g., "202501")
  • SMT Type - Must contain: Cross-sell, Lead Referral, or Testimonial/Compliment
  • SMT Orginator - Staff ID is extracted from first 8 characters

Logic: Counts records by Staff ID + Month for the three SMT types.

File 2: CME Client Engagement

Required Columns:

  • Attendee Employee ID - Staff ID (text/number)
  • Created By Employee ID - Staff ID (text/number)
  • Year & Month - Format: 5 digits (e.g., "20253" â†’ "202503")
  • Meeting Type - Values: Client Meeting, Service Review

Logic:

  • Client Meeting: Count by Attendee Employee ID + Month
  • Service Review: Count by Created By Employee ID + Month

File 3: CME PG

Required Columns:

  • Deal TL PS ID - Staff ID
  • Month&Year - Format: 5 digits (e.g., "20253")
  • Staff Team Country - Must be "China"
  • Domestic Contribution - Must equal 100
  • Deal Category_Alt - Must be "Won"
  • EBV (USD000) - Numeric value

Logic: For records matching ALL three filters (China + Won + Domestic=100):

  • Count records by Staff ID + Month
  • Sum EBV values by Staff ID + Month

Data Merging

Data from all three files is merged using Staff ID + Month as the key. Missing values default to 0.

Output

Column Description
Staff ID Employee identifier
Month Period (YYYYMM)
Cross-sell Count from SMT file
Lead Referral Count from SMT file
Testimonial Count from SMT file
Client Meeting Count as attendee from CME Client Engagement
Service Review Count as creator from CME Client Engagement
Deal Count Number of deals from CME PG
EBV Total Sum of EBV values from CME PG

How to Use

  1. Open the tool in your web browser
  2. Upload all three CSV files (any order)
  3. Click "Start Processing"
  4. Review the processing log (optional)
  5. View statistics and trend charts
  6. Select months and click "Export Selected Months" to download results
Note: The table displays only the top 20 records, but the export function includes ALL data.

Chart Features

  • 7 trend charts available (click tabs to switch)
  • Data aggregated by month across all staff
  • Line charts show trends over time

Common Issues

Q: Why are some metrics zero for my staff?

A: This means no data was found for that staff ID in that month for that metric.

Q: Why is my CME PG data count low?

A: CME PG applies strict filters. Records must meet ALL three conditions:

  • Staff Team Country = "China" (exact match)
  • Domestic Contribution = 100 (exact match)
  • Deal Category_Alt = "Won" (exact match)

Q: Where is all my data? I only see 20 rows.

A: For performance, only 20 rows display on screen. Use the export function to get all data.
